# 🏙️ 경기도 신도시 GS25 출점 전략 분석 대시보드

이 프로젝트는 경기도 내 주요 신도시(판교, 고덕, 광교)의 국민연금 데이터, 지하철 승하차량, 상업용 임대료 등 공공 데이터를 분석하여 **GS25의 최적 출점 전략**을 제안하는 인터랙티브 대시보드입니다.

## 🚀 주요 기능
- **시나리오별 다각도 분석**: 
  - **Scenario A (Demand Focus)**: 배후 인구 및 유동량 중심 입지 선정.
  - **Scenario B (Efficiency Focus)**: 임대료 대비 수요(가성비) 최적 지역 도출.
  - **Scenario C (Blue Ocean)**: 신규 택지 및 경쟁 밀도 분석을 통한 선점 전략.
- **인터랙티브 공간 분석**: Folium 기반 지역별 입지 매력도 시각화.
- **비즈니스 인사이트**: 지역 페르소나별 맞춤형 PB 상품(카페25, 도시락, 밀키트 등) 추천.
- **정교한 입지 유효성 검증**: 국민연금 가입자 기반 오피스 인구 가중치 적용 및 버거지수(공급지수) 활용.

## 📂 프로젝트 구조
- `app.py`: Streamlit 대시보드 메인 애플리케이션.
- `requirements.txt`: 프로젝트 실행을 위한 Python 패키지 의존성.
- `analysis_result_final.csv`: 최종 가공 및 스코어링이 완료된 분석 데이터.
- `reports/`: 심화 전략 보고서 (Markdown/PDF/PPTX).
- `github/`: 오픈소스 배포를 위한 가이드 및 데이터 세트.

## 🛠️ 실행 방법

1. **가상환경 활성화 및 패키지 설치**:
   ```bash
   uv pip install -r requirements.txt
   ```

2. **스트림릿 서버 실행**:
   ```bash
   streamlit run app.py
   ```

## 🎯 분석 대상 지역
- **판교 (삼평동, 백현동)**: IT 직장인 중심의 압도적 배후 수요.
- **평택 고덕 (고덕동)**: 삼성전자 평택캠퍼스 기반의 폭발적 성장 잠재지.
- **수원 광교 (광교1동)**: 고소득 전문직 및 거주 인구가 조화된 완성형 택지.
