# 경기도 신도시 GS25 출점 전략 분석 대시보드

이 프로젝트는 경기도 내 주요 신도시(판교, 고덕, 광교)의 부동산 데이터와 유동인구 데이터를 분석하여 GS25의 최적 출점 전략을 제안하는 스트림릿 기반 대시보드입니다.

## 🚀 주요 기능
- **시나리오별 분석**: 수요 중심(A), 효율 중심(B), 블루오션(C) 총 3가지 시나리오에 따른 입지 순위 제공.
- **인터랙티브 지도**: Folium을 활용하여 각 행정동별 입지 매력도를 시각화.
- **비즈니스 인사이트**: 타겟 페르소나와 그에 맞는 PB 상품 추천 기능.
- **데이터 분석 보고서**: 변수 유효성 검증 및 최종 합의안(Golden Spot) 도출.

## 📂 프로젝트 구조
- `app.py`: 스트림릿 메인 앱.
- `requirements.txt`: 필수 패키지 목록.
- `outputs/`: 분석 결과 데이터 (`analysis_result_final.csv` 포함).
- `reports/`: 심화 전략 보고서 (Markdown 포맷).
- `source/`: EDA 결과 및 PDF 보고서.
- `process_csv.py`, `optimize_filter.py`: 데이터 전처리 및 분석 스크립트.

## 🛠️ 실행 방법

1. 필수 패키지 설치:
   ```bash
   pip install -r requirements.txt
   ```

2. 스트림릿 서버 실행:
   ```bash
   streamlit run app.py
   ```

## 📊 분석 대상 지역
- **판교 (삼평동, 백현동)**: 배후 수요가 압도적인 IT 허브.
- **평택 고덕 (고덕동)**: 삼성 반도체 배후의 고성장 블루오션.
- **수원 광교 (광교1동)**: 가족 단위 거주자와 전문직이 혼재된 완성형 신도시.
